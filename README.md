# Contact Management System

## 📌 Project Overview
The **Contact Management System** is a simple web-based application developed using **HTML, CSS, and JavaScript**. The purpose of this project is to help users manage contact information efficiently and systematically.

Users can:
- Add new contacts
- Update existing contact details
- Delete contacts
- Search and manage contacts easily

This project also demonstrates practical implementation of **GitHub tools and workflows**, including repository management, branching, pull requests, merging, and GitHub Actions.

---

## 🚀 Features

### 1. Add Contact
Users can add new contacts by entering details such as name, phone number, email, and address.

### 2. Update Contact Information
Users can edit and update existing contact details whenever required.

### 3. Delete Contact
Users can remove unwanted or outdated contacts from the contact list.

### 4. Search Contacts
Users can search contacts quickly by name or phone number.

### 5. View Contact List
Displays all saved contacts in an organized format.

### 6. User-Friendly Interface
Simple and clean interface for easy contact management.

### 7. Real-Time Contact Management
Changes are updated instantly without refreshing the page.

### 8. Lightweight Application
Simple, fast, and easy to use.

### 9. Organized Contact Handling
Helps users store and manage contact details efficiently.

### 10. GitHub Collaboration Support
This project demonstrates:
- Repository management
- Branching
- Pull requests
- Collaboration workflow

### 11. GitHub Actions Integration
A simple GitHub Actions workflow is implemented for automation and Continuous Integration (CI).
